# 🚀 NexusChat — Real-Time Chat Application

A modern **real-time chat platform** built using **WebSocket technology** that enables instant messaging between users with chat rooms, media sharing, and live presence indicators.

This project was developed as part of my **Full-Stack Development Internship at Prodigy InfoTech (Task-04)**.

The application simulates a **production-style chat platform** where users can create accounts, join rooms, exchange messages instantly, and manage conversations through an intuitive and interactive dashboard.

---

# ✨ Features

## 💬 Core Chat Features

* Real-time messaging using **WebSockets (Socket.IO)**
* Create and join **chat rooms**
* Private and group conversations
* Instant message delivery with **live updates**
* Message timestamps and chat history
* Emoji support
* Media/image sharing
* Message input with multiline support

---

## 🔔 Advanced Features

* **Live user presence indicators** (online/offline)
* **Real-time notifications**
* **Chat history persistence**
* Room member management
* Media attachments in messages
* Auto-scroll chat updates
* Typing indicators
* Responsive conversation UI

---

## 👤 Authentication & Security

* Secure **User Registration and Login**
* **JWT Authentication**
* Password hashing using **bcrypt**
* Protected API routes
* Session handling
* Role-based access (Admin / User)

---

## 🛠 Admin Dashboard

# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/keerthanaAR25/PRODIGY_FS_04.git
```

---

### 2️⃣ Install Backend Dependencies

```
cd backend
npm install
```

---

### 3️⃣ Install Frontend Dependencies

```
cd ../frontend
npm install
```

---

### 4️⃣ Run the Backend Server

```
cd backend
npm run dev
```

---

### 5️⃣ Run the Frontend

```
cd frontend
npm run dev
```
